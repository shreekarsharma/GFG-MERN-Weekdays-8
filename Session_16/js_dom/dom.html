DOM

Document Object Model

DOM is a programming interface for web documents.

It represents the structure of a document as a tree of objects.

Each object corresponds to a part of the document.

DOM allows programming languages to manipulate the document.

It provides a way to access and modify the content, structure, and style of a document.

DOM is language-agnostic, but most commonly used with JavaScript.

DOM is a W3C standard.

DOM is supported by all modern browsers.

DOM is a core technology of the web.

DOM is used to create dynamic web pages.
